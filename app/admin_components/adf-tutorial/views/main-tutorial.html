• Create Service
    This is where you would want to start. Services are the building blocks of DreamFactory. They are the entrypoints for interaction with your application.     Sometimes you would want to add the custom behavior to some of your endpoint. To achieve that we have this feature called Scripting. For example this interactive guide will show you how to add the custom PHP code to one of the standard system endpoints.
